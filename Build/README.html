Trigger

WTF

Trigger is a lightweight HTML5 template made to use with Hammer for Mac. If Hammer isn’t your thing, I’ve branched off a non-hammer version of this.

It’s based on the HTML5 Boilerplate, with some of the guff I never use stripped out, and some SASS & Bourbon Mixins added in.
